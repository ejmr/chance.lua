# Change Log

This document describes notable changes to this project, which adheres
to [Semantic Versioning](http://semver.org/).



## Unreleased

### Added

- A basic test suite.

- The `Core` API:
    - `chance.VERSION`
    - `chance.seed()`
    - `chance.random()`

- The `Basic` API:
    - `chance.bool()`
    - `chance.float()`
    - `chance.integer()`
    - `chance.natural()`
    - `chance.character()`

- The `Helper` API:
    - `chance.pick()`
    - `change.shuffle()`

- The `Miscellaneous` API:
    - `chance.rpg()`
    - `chance.d4()`
    - `chance.d6()`
    - `chance.d8()`
    - `chance.d10()`
    - `chance.d12()`
    - `chance.d20()`
    - `chance.d100()`
    - `chance.n()`
